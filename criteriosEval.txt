
REQUISITOS:

1. El sitio web debe poseer al menos 4 páginas HTML o 4 secciones si se trata de un sitio One Page. 
    ver: home, contact, offers, offices.

2. El sitio web debe incluir un formulario de contacto, con al menos 5 campos que incluya (un checkbox o radiobutton), un select y una imagen, con validación realizada mediante JavaScript para que los campos sean obligatorios. 
    ver: contact.html 
    posee: una imagen de fondo, un select, checkbox, label, input.
    mediante JS: se verifican los campos obligatorios y se muestra una alerta si hay alguno incompleto.
    los campos de input tienen patrones (pattern) para validar el tipo de datos ingresados.


3. Tener al menos una página totalmente responsive con mínimo 3 puntos de corte con media querys para 3 tamaños de dispositivos (PC escritorio, Tablet, Celular). Para el diseño debe usarse CSS.
    todo el sitio es responsive para dos medidas (@media screen): menores de 600px y mayores de 601px

4. Utilizar al menos una animación, transformación o una transición.
    - en el menú de navegación los botones tienen animación, en especial el de 'ofertas'.

5. Poseer una estructura HTML maquetada con Flexbox y/o Grid.
    ver: los archivos css

6. Se considerará la presentación general del proyecto, la legibilidad del mismo, la navegación sin llegar a puntos de no retorno, la optimización de imagenes para la web y el uso de favicon
    - imagenes optimizadas
    - uso de favicon.

7. El trabajo práctico deberá subirse a un servidor online y compartirse mediante un repositorio de Git. 

8. La página deberá subirse a un servidor on-line para poder ser navegada por el Docente. Ejemplo: Netlify o similar. 
    - disponible en

9. El sitio web debe estar estructurado utilizando etiquetas semánticas HTML correctamente. Debe pasar la validación de https://validator.w3.org/ sin errores. 

10.Utilizar iframes y/o íconos de FontAwesome y/o fuentes locales o bien de Google Fonts. Es optativo incluir algún elemento de Bootstrap.
